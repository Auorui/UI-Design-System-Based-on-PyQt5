## BatchVision 批处理系统使用手册

### 🌟 系统概述
一款基于PyQt5开发的桌面端批处理工具，集成三大核心功能：
- **图像尺寸批量修改**（保持比例/强制缩放）
- **智能数据集划分**（训练集/验证集/测试集）
- **高级重命名系统**（带序号生成/格式转换）
- **扩展功能**：灰度化处理、图像统计（均值/标准差）

详见[BatchVision.md](./instructions/BatchVision.md)
